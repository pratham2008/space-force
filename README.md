# SPACE FORCE

A cinematic arcade space shooter built with Flutter and Flame.

## Features
- Fast-paced arcade gameplay
- Cinematic visual effects and animations
- Parallax starfield background
- Layered neon UI effects
- Dynamic wave transitions
- Specialized audio system with background music and sound effects
